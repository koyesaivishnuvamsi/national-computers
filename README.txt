# National Computers Website

This is a responsive Angular + Tailwind CSS website for a computer store.

## 🚀 Getting Started

1. Install Angular CLI (if not installed):
   npm install -g @angular/cli

2. Install dependencies:
   npm install

3. Run locally:
   ng serve

4. Build for production:
   ng build --prod

## 🧩 Project Structure

- Navbar and Footer shared across pages
- Pages: Home, Laptops, Desktops, Gaming Setup, Accessories, Services, Contact Us
- Product data: `src/assets/products.json`

## 🌐 Deployment (Recommended: Vercel)

1. Push this code to a GitHub repository
2. Go to https://vercel.com and sign in with GitHub
3. Click **"New Project"**, import your repo, and deploy

Vercel auto-detects Angular and builds automatically.
